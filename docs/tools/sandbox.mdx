---
title: "沙箱工具"
description: "Strix 容器中预装的安全工具"
---

Strix 运行在基于 Kali Linux 的 Docker 沙箱中，容器内预装了较完整的安全工具集。代理可以通过[终端](/tools/terminal)直接调用这些工具。

## Reconnaissance

| 工具 | 说明 |
|------|------|
| [Subfinder](https://github.com/projectdiscovery/subfinder) | 子域发现 |
| [Naabu](https://github.com/projectdiscovery/naabu) | 高速端口扫描 |
| [httpx](https://github.com/projectdiscovery/httpx) | HTTP 探测与分析 |
| [Katana](https://github.com/projectdiscovery/katana) | Web 爬取与 spider |
| [ffuf](https://github.com/ffuf/ffuf) | 高速 Web fuzzing |
| [Nmap](https://nmap.org) | 网络扫描与服务识别 |

## Web 测试

| 工具 | 说明 |
|------|------|
| [Arjun](https://github.com/s0md3v/Arjun) | HTTP 参数发现 |
| [Dirsearch](https://github.com/maurosoria/dirsearch) | 目录与文件爆破 |
| [wafw00f](https://github.com/EnableSecurity/wafw00f) | WAF 指纹识别 |
| [GoSpider](https://github.com/jaeles-project/gospider) | 提取链接的 Web spider |

## 自动化扫描

| 工具 | 说明 |
|------|------|
| [Nuclei](https://github.com/projectdiscovery/nuclei) | 基于模板的漏洞扫描 |
| [SQLMap](https://sqlmap.org) | 自动化 SQL 注入检测与利用 |
| [Wapiti](https://wapiti-scanner.github.io) | Web 应用漏洞扫描 |
| [ZAP](https://zaproxy.org) | OWASP Zed Attack Proxy |

## 源码感知分析
