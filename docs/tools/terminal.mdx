Strix 可以使用运行在 Docker 沙箱内的持久 Bash 终端，因此能够直接调用所有[预装安全工具](/tools/sandbox)。

## 能力概览

| 能力 | 说明 |
|------|------|
| 持久状态 | 工作目录与环境变量会在命令之间保留 |
| 多会话 | 可并行开启多个终端处理不同任务 |
| 后台任务 | 可启动长时间运行的进程而不阻塞主流程 |
| 交互控制 | 可响应提示并控制运行中的进程 |

## 常见用途

### 运行安全工具

```bash
# 子域枚举
subfinder -d example.com

# 漏洞扫描
nuclei -u https://example.com

# SQL 注入测试
sqlmap -u "https://example.com/page?id=1"
```

### 代码分析

```bash
# 快速 SAST 初筛
semgrep --config auto ./src

# 结构化 AST 搜索
sg scan ./src

# Secret 检测
gitleaks detect --source ./
trufflehog filesystem ./

# 供应链和配置检查
trivy fs ./
```

### 自定义脚本

```bash
# 执行 Python 利用脚本
python3 exploit.py

# 执行 Shell 脚本
./test_auth_bypass.sh
```

## 会话管理

代理可以同时维护多个终端会话，例如：

- 主会话用于核心测试
- 次会话用于监控或旁路检查
- 后台进程用于临时服务、watcher 或长任务
