# 020 公开文档汉化

## 目标

完成 README、贡献文档与 `docs/` 下直接面向用户和维护者的说明性页面汉化，统一站点口径。

## 范围

- `README.md`
- `CONTRIBUTING.md`
- `docs/advanced`
- `docs/llm-providers`
- `docs/tools`
- 其余直接面向用户的说明性页面

## 输入

- 010 术语规范
- 现有中文 README / docs 页面
- 当前站点导航结构

## 变更点

- 标题、段落、表格说明、注意事项、示例上下文统一改为中文
- 保留命令、配置键、模型 ID、环境变量和代码块不翻译
- 统一模型文档口径：优先说明兼容网关、本地模型和国内可达方案
- 统一工具文档口径：说明当前能力、使用方式与兼容工作流

## 验收条件

- 文档页面不再出现大段说明性英文段落
- 导航、标题、简介、示例说明风格一致
- 保留的英文仅限术语白名单与技术字面量

## 剩余问题

- 新增页面进入仓库时，仍需按 050 的治理规则继续检查
