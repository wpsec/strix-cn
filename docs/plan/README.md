# 全面汉化计划索引

本文档用于集中管理 `strix-cn` 的汉化推进计划。后续所有计划文档统一放在 `docs/plan` 目录下，采用 `NNN-主题.md` 的编号形式维护。

## 目标

- 建立简体中文默认体验，覆盖用户可见面与开发文档
- 保持命令、配置键、模型 ID、协议字段和其他技术字面量不变
- 不引入新的 i18n 框架，优先直接收口现有中文文案
- 通过测试和残留审计，避免后续回退到英文

## 推进原则

- 计划文档只放在 `docs/plan`
- 所有新增用户可见文案默认使用简体中文
- 允许保留的英文仅限白名单内容，例如环境变量、命令、模型名、CVE/CWE、HTTP 头、代码块、JSON 键、上游产品名
- 同一术语在 README、docs、CLI、TUI、报告中保持统一译法
