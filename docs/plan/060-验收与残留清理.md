# 060 验收与残留清理

## 目标

通过残留审计、测试和冒烟，把本轮汉化的收尾标准固定下来，并记录允许保留的英文项。

## 范围

- `README.md`
- `CONTRIBUTING.md`
- `docs/`
- `strix/interface/`
- 用户可见的 `strix/tools/` 渲染与提示代码
- 相关测试

## 输入

- 010 术语规范与英文白名单
- 已完成的文档、CLI、TUI、提示词改动
- 冒烟输出与测试结果

## 变更点

- 按关键目录执行英文残留审计
- 对残留内容应用白名单，避免误判技术字面量
- 补齐 CLI/TUI/报告中文输出的关键测试
- 记录仍保留英文的原因和位置

## 验收条件

- 用户主路径中仅剩白名单英文
- 文档、运行提示、默认报告语言一致
- 新增测试可以防止高频文案回退到英文

## 当前白名单示例

- 模型 ID、环境变量、命令、JSON 键
- 第三方产品名与厂商名
- 第三方库原始错误
- 代码块、payload、请求报文、协议字段
