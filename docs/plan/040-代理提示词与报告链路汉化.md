# 040 代理提示词与报告链路汉化

## 目标

调整默认提示词与报告相关工具说明，降低运行过程中的英文旁白概率，并继续确保最终报告默认中文输出。

## 范围

- `strix/agents/prompts/system_prompt.jinja`
- 影响默认输出语言的工具说明
- `finish_scan`、报告创建工具、最终执行摘要链路

## 输入

- 当前系统提示词
- 报告工具现有中文约束
- 用户实际运行时的英文旁白表现

## 变更点

- 显式声明默认用户可见输出语言为简体中文
- 保持最终报告、漏洞描述、依赖漏洞描述默认中文
- 明确保留英文的技术字面量范围，如 PoC、请求报文、代码和 payload
- 用户显式要求英文或双语时，允许通过 instruction 覆盖默认行为

## 验收条件

- 默认运行时的旁白不再明显偏向英文
- 最终报告标题和正文继续保持中文
- 用户显式要求英文时，不影响覆盖能力

## 剩余问题

- 具体输出仍受模型与网关能力影响，需要结合 060 冒烟结果持续迭代
