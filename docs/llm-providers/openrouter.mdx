---
title: "OpenRouter"
description: "通过 OpenRouter 为 Strix 配置多家模型"
---

[OpenRouter](https://openrouter.ai) 通过单一 API 统一接入多家模型提供商，适合希望快速切换模型或做路由冗余的场景。

## 配置

```bash
export STRIX_LLM="openrouter/openai/gpt-5.4"
export LLM_API_KEY="sk-or-..."
```

## 可用模型

OpenRouter 采用 `openrouter/<provider>/<model>` 格式：

| 模型 | 配置 |
|------|------|
| GPT-5.4 | `openrouter/openai/gpt-5.4` |
| Claude Sonnet 4.6 | `openrouter/anthropic/claude-sonnet-4.6` |
| Gemini 3 Pro | `openrouter/google/gemini-3-pro-preview` |
| GLM-4.7 | `openrouter/z-ai/glm-4.7` |

## 获取 API Key

1. 打开 [openrouter.ai](https://openrouter.ai)
2. 登录后进入 Keys
3. 创建新的 API key

## 特点

- **单一 API**：统一访问 OpenAI、Anthropic、Google、Meta 等模型
- **故障切换**：可在不同提供商间做 fallback routing
- **成本汇总**：统一统计多模型使用成本
- **速率更灵活**：由 OpenRouter 处理部分提供商限流问题
