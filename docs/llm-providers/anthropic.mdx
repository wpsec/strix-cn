## 配置

```bash
export STRIX_LLM="anthropic/claude-sonnet-4-6"
export LLM_API_KEY="sk-ant-..."
```

## 可用模型

| 模型 | 说明 |
|------|------|
| `anthropic/claude-sonnet-4-6` | 智能与速度平衡较好的默认选择 |
| `anthropic/claude-opus-4-6` | 更适合深度分析的高能力模型 |

## 获取 API Key

1. 访问 [console.anthropic.com](https://console.anthropic.com)
2. 进入 `API Keys`
3. 创建新的 key
