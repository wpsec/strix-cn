## 安装

Vertex AI 需要额外的 Google Cloud 依赖。请使用带 `vertex` extra 的安装方式：

```bash
pipx install "strix-agent[vertex]"
```

## 配置

```bash
export STRIX_LLM="vertex_ai/gemini-3-pro-preview"
```

无需额外 API key，默认使用 Google Cloud Application Default Credentials。

## 认证方式

### 方式 1：gcloud CLI

```bash
gcloud auth application-default login
```

### 方式 2：Service Account

```bash
export GOOGLE_APPLICATION_CREDENTIALS="/path/to/service-account.json"
```

## 可用模型

| 模型 | 说明 |
|------|------|
| `vertex_ai/gemini-3-pro-preview` | 更适合高质量安全测试 |
| `vertex_ai/gemini-3-flash-preview` | 更快、更便宜 |

## 项目配置

```bash
export VERTEXAI_PROJECT="your-project-id"
export VERTEXAI_LOCATION="global"
```

## 前置条件

1. 在 Google Cloud 项目中启用 Vertex AI API
2. 确保当前账号具备 `Vertex AI User` 角色
