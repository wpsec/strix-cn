title: "OpenAI"
description: "为 Strix 配置 OpenAI 模型"
---

## 配置

```bash
export STRIX_LLM="openai/gpt-5.4"
export LLM_API_KEY="sk-..."
```

## 可用模型

完整模型列表请参考 [OpenAI Models Documentation](https://platform.openai.com/docs/models)。

## 获取 API Key

1. 打开 [platform.openai.com](https://platform.openai.com)
2. 进入 API Keys
3. 创建新的 secret key

## 兼容 base URL

如果你接入的是 OpenAI-compatible 网关或代理服务，也可以保留 `openai/...` 形式并改写 `LLM_API_BASE`：

```bash
export STRIX_LLM="openai/gpt-5.4"
export LLM_API_KEY="your-key"
export LLM_API_BASE="https://your-proxy.com/v1"
```
